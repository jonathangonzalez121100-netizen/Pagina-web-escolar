# Página Web Escolar 

Proyecto de una página web educativa desarrollada con tecnologías web básicas, enfocada en una navegación clara, diseño responsive y buenas prácticas de frontend.

---

##  Demo
https://jonathangonzalez121100-netizen.github.io/Pagina-web-escolar/

---

## Características
- Barra de navegación responsive
- Cambio de estilo en la navbar al hacer scroll
- Diseño adaptable a dispositivos móviles
- Uso de variables CSS para fácil personalización
- Estructura semántica HTML5
- Implementación de Formulario (aún no funcional)

---

## Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox, Media Queries, Variables CSS)
- JavaScript (DOM, eventos, scroll)
- Git & GitHub
